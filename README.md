🚀 How to Run

Clone the repository:

git clone https://github.com/your-username/selenium-4x-learning.git
cd selenium-4x-learning


Install dependencies:

mvn clean install

Run tests:
mvn test

Generate Allure Report:

allure serve target/allure-results
